本项目致力于打造无需法官的线下狼人杀体验。

在黑夜期间，每个存活的玩家都需要执行所有环节的投票，例如狼人刀人，女巫救人、预言家验人，守卫守人等，但只有对应身份玩家的投票才会生效。

每个环节需要所有人都投票后，才可以点击下一步，完成所有环节后，即可天亮。

## 项目结构

由前端项目 client 和后端项目 server 组成，主要内容如下：

```
├── client
│   ├── src
│   │   ├── Login  // 登录页面
│   │   ├── Waiting // 等待开始游戏
│   │   ├── Night // 夜晚
│   │   ├── Daytime // 白天

├── server
│   ├── src
│   │   ├── room  // 房间信息
│   │   ├── player // 玩家信息
```
